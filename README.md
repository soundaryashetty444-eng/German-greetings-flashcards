# German Greetings Flashcard Web App

An interactive web application designed to help beginners learn and remember basic German greetings using flashcards, along with a built-in quiz feature to test understanding.

##  Features
- Flashcard-based learning for German greetings
- “Test Yourself” quiz with multiple-choice questions
- Clean and user-friendly interface
- Smooth UI interactions and animations
- German-themed visual design

##  Technologies Used
- HTML
- CSS
- JavaScript

##  Project Purpose
This project was created to strengthen foundational frontend development skills, improve JavaScript logic, and understand real-world project deployment workflows.
